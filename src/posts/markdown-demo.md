## 3. 代码

我们还可以添加代码，支持代码高亮：

```javascript
//javascript
export default function Button() {
  const handleClick = () => console.log("Button Clicked");

  return (
    <div onClick={handleClick} className="py-2 px-3 rounded-md bg-indigo-600 text-white">
      Click Me
    </div>
  );
}
```
